# Killswitch

Next-generation on-chain security terminal with real-time threat detection and automated vault circuit breakers.

## Features
- **Real-time Heuristics**: Deep analysis of transaction patterns.
- **Auto-Guardian**: Automated freezing of vaults when threats are detected.
- **Incident Reports**: Detailed traces and resolution workflows.
- **Web3 Integration**: Native wallet support and multisig protocols.

## Tech Stack
- React + Vite
- Tailwind CSS + Framer Motion
- Recharts
- Wagmi + Viem
- RainbowKit
